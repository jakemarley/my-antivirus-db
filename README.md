
# My Antivirus
A lightweight antivirus scanner written in Python featuring real-time protection, signature detection, and quarantine management.




## Features

- Quick Scan
- Full Scan
- Scan Folder
- Scan Running Processes
- Real-Time Protection
- Quarantine Manager
- Virus Signature Updates
- Software Version Check
- Activity Log
- Scan Progress Bar
  

Download the latest version here:
https://github.com/jakemarley/my-antivirus-db/releases

## Installation

1. Download the latest release
2. Extract the zip file
3. Run `MyAntivirus.exe`
4. Click **Update Signatures** after starting
