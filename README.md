
# My Antivirus
A lightweight antivirus scanner written in Python with real-time protection, 
signature detection, and quarantine management.



Notes:
Windows may show a security warning because the program is not digitally signed yet.
Click "More info" → "Run anyway".


## Features

• Quick Scan
• Full Scan
• Scan Folder
• Scan Running Processes
• Real-Time Protection
• Quarantine Manager
• Virus Signature Updates
• Software Version Check
• Activity Log
• Scan Progress Bar
